📥 Come Clonare e Lanciare il Progetto

Segui questi semplici passaggi:


1️⃣ Scarica Git e Node.js
Assicurati di avere installato:

Git: Scaricalo da https://git-scm.com/

Node.js: Scaricalo da https://nodejs.org/


2️⃣ Clona il Progetto

Vai alla pagina del progetto su GitHub e clicca sul pulsante verde "Code" e copia il link HTTPS.

Nel terminale, digita:


git clone URL_DEL_PROGETTO

3️⃣ Apri i Terminali

Apri due terminali separati:

Primo Terminale:


cd frontend

Secondo Terminale:

cd server

Rinomina i terminali con F2 per evitare confusione.

4️⃣ Installa le Dipendenze

Nel frontend, esegui:


npm install @fortawesome/react-fontawesome @fortawesome/free-solid-svg-icons react-icons react-router-dom react-bootstrap


Nel backend, esegui:

npm install jsonwebtoken mongoose bcryptjs express multer path @sendgrid/mail cors dotenv

5️⃣ Avvia il Progetto

Nei terminali (frontend e backend), esegui:

npm run dev
Apri il browser e visita il link che appare (es. http://localhost:3000) per vedere il progetto in azione! 🎉
